Concert Harp samples for [Sanic Boom](https://sanicboom.com).

## Samples

45 samples available. See `library.json` for the full alias list.

## Usage

```sanic
import "github:SanicBoom/concert-harp"

let snd = sample("lib:concert-harp/a2_f1")
play snd
```
